Class Delta

A helper class returned upon retrieving changes from the directory server

Instances of this class represent a current state of a search query for which you would later like to retrieve changes. You should always serialize these instances and store them for later use in order for the change tracking to work.

Namespace: ADX\Core
See: ADX\Core\Task::changes() Pull changes from Active Directory for a given query
