Class Selector

Abstract class to provide unified behaviour for classes that list ( select ) specific items from Active Directory

Subclass the Selector to create a simple, easy to use search task for your code. While subclassing, you only need to provide the operation to be performed ( scope subtree by default ) and the ldap filter to be executed. Then, in your code, you only need to instantiate your Selector and call ADX\Util\Selector::all() on it to get all the objects that match the search query. You can also perform additional processing by implementing the self::_process_result() method.
